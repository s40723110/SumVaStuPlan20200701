七月五號
(接續七月四號主題)

為了瞭解運動矩之間的向量(包括不同平面間 ) 的關係，參考 : 

https://www.youtube.com/watch?v=x7tkStus80U&feature=youtu.be
----

一開始在琢磨 line 31 行 :

t = (f-e) * tan30 / 2.0

一開始不知道此行的 tan30 / 2 是怎麼來的， 參考上面影片後，發現有一平面向量與其他力臂向量在 " 不同平面 "，所以需要用轉換矩陣至不同平面上的方程式 轉換成同一平面上的向量方可推導公式 ， 所以此動作為帶入矩陣轉換方程式 。
